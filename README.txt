WELCOME:

WordSmith is a service that takes a user input of up to two words, and returns a list of
5 synonyms for that word, and sentences that explain how each of those words might be used.
The purpose of this service is to help someone who wants to choose the word that best matches 
the thoughts that they want to convey. This tool can help people who can't quite remember 
a word they read in a book, or English learners who dont' have quite as wide of a vocabulary. 

USER'S MANUAL:

1 - run backend.py, will start app as local device as host. 
2 - go to internet browser of choice and navigate to http://127.0.0.1:5000/WordSmith
3 - input up to two words in the input box
4 - click "Generate Synonyms"
